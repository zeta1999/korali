*******************************

Laplace
-------
          
The Laplace distribution has the probability density function (PDF):

.. math::

    f(x;\mu ,b)=\frac{1}{2b}exp(-\frac{|x-\mu|}{b})
    
where :math:`\mu` is a the **mean** parameter and :math:`b > 0` is the **scale** parameter.
